# Code of Conduct

OpenAN is a Linux Foundation Networking project. All participants in the OpenAN community are bound by the [LF Projects Code of Conduct](https://lfprojects.org/policies/code-of-conduct/). By participating, you are expected to uphold this code.

If you experience or witness unacceptable behavior, or have any other concerns, please report it by contacting the project maintainers or by emailing the [Linux Foundation](mailto:conduct@lfprojects.org).
