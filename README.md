# .github

This repository contains the default [community health files](https://docs.github.com/en/communities/setting-up-your-project-for-healthy-contributions/creating-a-default-community-health-file) for the [OpenAN](https://github.com/project-openan) organization.

These files apply as defaults to all repositories in the `project-openan` organization that do not have their own versions.

## Contents

| File | Purpose |
|------|---------|
| `profile/README.md` | Organization profile displayed on [github.com/project-openan](https://github.com/project-openan) |
| `CODE_OF_CONDUCT.md` | Community Code of Conduct (references [LF Projects CoC](https://lfprojects.org/policies/code-of-conduct/)) |
| `CONTRIBUTING.md` | Contribution guidelines, DCO sign-off requirements, and SPDX header guidance |
| `SECURITY.md` | Security vulnerability reporting policy |
| `SUPPORT.md` | Where to find help and community resources |
| `PULL_REQUEST_TEMPLATE.md` | Default pull request template with DCO checklist |
| `ISSUE_TEMPLATE/bug_report.yml` | Bug report issue form |
| `ISSUE_TEMPLATE/feature_request.yml` | Feature request issue form |
| `ISSUE_TEMPLATE/config.yml` | Issue chooser configuration (links to mailing lists, security policy) |
| `LICENSE` | Apache License 2.0 |
